## The Constraint: Distribution vs Conversion

**Framework:**
- If traffic < 1,000/month → optimize **distribution**
- If traffic > 1,000/month → optimize **conversion**

**Current decision:** We're playing **DISTRIBUTION**

### What This Means

✅ **Do:**
- Create shareable content ("Signer of Record Gap" post)
- Build unavoidable demos (pp-demo repo)
- Direct outreach to AI-native startups
- Get indexed by search engines
- Post on Twitter/HN/Reddit
- Speak at meetups/podcasts

❌ **Don't:**
- A/B test CTA button colors
- Optimize form field count
- Add exit-intent popups
- Obsess over bounce rate
- Build elaborate onboarding funnels

### Why

You can't optimize conversion when nobody's visiting. Every minute spent on conversion optimization at <1,000 visitors/month is wasted. Focus on getting eyeballs first.
